# Sistema de Oficina - WEG (Recuperação)

## 📌 Sobre o projeto
Este projeto é um sistema de oficina desenvolvido utilizando **Java com Spring Boot**.  
O objetivo é gerenciar informações relacionadas à oficina, como cadastro e controle de dados.

## 🚀 Tecnologias utilizadas
- Java 21
- Spring Boot
- Spring Data JPA
- Maven

## 📂 Estrutura do projeto
O projeto segue uma organização padrão:
- `model` → entidades do sistema  
- `repository` → acesso ao banco de dados  
- `service` → regras de negócio  
- `controller` → controle das requisições  

## ▶️ Como executar
1. Clone o repositório
2. Abra o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
3. Execute a classe principal (Spring Boot)
4. A aplicação estará disponível em:

http://localhost:8080


## 🧪 Funcionalidades
- Cadastro de dados
- Listagem de informações
- Integração com banco de dados

## 📄 Observações
Este projeto foi desenvolvido para fins de estudo/recuperação.
